# 容器简介

容器采用插件为对应的元素进行了 class 的指定，然后在 .vuepress/styles/index.styl 中进行自定义对应类对应的样式
也就是安装插件后不会直接显示效果，但是对应的元素已经具有了特殊的 class，具体渲染效果还需自己指定

第一步先安装 vuepress-plugin-container 插件，这个插件可以在配置文件中多次使用

yarn add -D vuepress-plugin-container

# 文字居中

vuepress 实现文字居中

index.styl

.customer-container-center
  text-align center

config.js

    [
      'vuepress-plugin-container',
      { //添加居中容器
        type: 'center',
        before: info => `<div class="customer-container-center">`,
        after: '</div>',
      },
    ],

# 文字居右

要居右的段落

index.styl

.custom-block
  &.right
    color transparentify($textColor, 0.4)
    font-size 0.9rem
    text-align right

config.js

    [
      'vuepress-plugin-container',
      {
        type: 'right',
        defaultTitle: '',
      },
    ],

# details 容器

SHOWMORE

下拉容器

语法:

/::: showmore
/内容
/:::

config.js

    [
      'vuepress-plugin-container',
      { //添加 details 容器
        type: 'details',
        before: info => `<details class="custom-block details">${info ? `<summary>${info}</summary>` : ''}\n`,
        after: () => '</details>\n'
      },
    ],

不用添加 CSS/styl

假若施加于某物体的外力为零，则该物体的运动速度不变。

来自 [维基百科](https://zh.wikipedia.org/wiki/%E7%89%9B%E9%A1%BF%E8%BF%90%E5%8A%A8%E5%AE%9A%E5%BE%8B)

index.styl

.today
  margin 1rem 0
  padding .1rem 1.5rem
  border-radius 0.4rem
  background-color #f0f4f8
  .title
    font-weight bold

config.js

    [
      'vuepress-plugin-container',
      {
        type: 'today',
        before: info => `<div class="today"><p class="title">${info}</p>`,
        after: '</div>',
      },
    ],

# 示例 config.js

最终配置的插件部分 js 代码示例:

  plugins: [
    ['flowchart'],
    ['img-lazy'],
    ['code-switcher'], //多语言选项卡
    ["cursor-effects"], //鼠标特效
    ["vuepress-plugin-reading-progress"], //上方进度条
    ["vuepress-plugin-cat"], //🐱
    ['@vuepress/pwa', { //pwa
      serviceWorker: true,
      updatePopup: {
        message: "New Content！",
        buttonText: "Refresh"
      }
    }], //这是一个bug插件，去除后浏览器还是读取之前的缓存，加上之后就得一直留着了，，并且外观很丑
    //容器添加
    [
      'vuepress-plugin-container',
      {
        type: 'right',
        defaultTitle: '',
      },
    ],
    [
      'vuepress-plugin-container',
      {
        type: 'today',
        before: info => `<div class="today"><p class="title">${info}</p>`,
        after: '</div>',
      },
    ],
    [
      'vuepress-plugin-container',
      { //添加 details 容器
        type: 'details',
        before: info => `<details class="custom-block details">${info ? `<summary>${info}</summary>` : ''}\n`,
        after: () => '</details>\n'
      },
    ],
    [
      'vuepress-plugin-container',
      { //添加居中容器
        type: 'center',
        before: info => `<div class="customer-container-center">`,
        after: '</div>',
      },
    ],
    [
      'vuepress-plugin-container',
      { //添加居右容器
        type: 'right',
        defaultTitle: '',
      },
    ],
  ],
